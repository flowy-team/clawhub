# ClawhHub

A curated collection of MCP skills for Claude Code.

## What is ClawhHub?

ClawhHub is a registry for installing and managing MCP (Model Context Protocol) skills in Claude Code. Skills extend Claude Code with new tools — connecting it to external platforms like FlowyTeam, GitHub, Notion, and more.

## Installed Skills

| Skill | Version | Description |
|---|---|---|
| [flowyteam-mcp](skills/flowyteam-mcp/) | 1.1.0 | Manage FlowyTeam projects, tasks, OKRs, KPIs, HR, CRM, finance, and more via MCP |

## Installing a Skill

```bash
clawhub install <owner>/<skill-name>
```

Example:

```bash
clawhub install agungksidik/flowyteam-mcp
```

## Updating a Skill

```bash
clawhub update <skill-name>
```
